name: load-existing-campaign
description: Загрузка существующей кампании для продолжения игры
---

## Phase 1: Выбор кампании
1. Показать список доступных кампаний из `./history/[game_name]`
2. Пользователь выбирает кампанию

## Phase 2: Загрузка состояния
1. Загрузить последнюю сессию
2. Загрузить карточки персонажей `./history/[game_name]/characters/`
3. Вывести кратную информацию по состоянию текущих персонажей 
4. Описать кратко текущую сцену

## Phase 3: Продолжение игры
1. Загрузить в сессию промт агента `.claude/agents/gm.md` и продолжить игру строго согласно инструкциям.
