## 2. ПРОВЕРКИ И СЛОЖНОСТЬ

### Таблица сложностей

| Сложность | Модификатор |
|-----------|-------------|
| Очень легкая | +60 |
| Легкая | +40 |
| Средняя | +20 |
| **Серьезная** | **+0 (без модификатора)** |
| Сложная | -10 |
| Трудная | -20 |
| Очень трудная | -30 |

### Типы проверок

- **Простые проверки:** d100 против умения. Успех = результат ≤ значению умения
- **Драматические проверки:** Важен уровень успеха (УУ)
- **Длительные проверки:** Требуют накопления определенного количества УУ за несколько бросков
- **Уровни успеха (УУ):** Разница между десятками умения (после модификаторов) и десятками на кубике

### Помощь

- Один персонаж совершает проверку
- Каждый помощник дает **+10**
- Помощник должен иметь хотя бы 1 шаг развития в навыке и находиться рядом

## 3. ПРЕИМУЩЕСТВО (ADVANTAGE)

### Эффект
- Каждое преимущество дает **+10** к боевым или психическим проверкам

### Получение преимущества
- Атака ошеломленного врага
- Использование натиска
- Победа во встречной проверке
- Нанесение ран без вступления в бой

### Потеря преимущества
- Провал встречной проверки
- Получение любого состояния
- Получение раны

## 4. БОЙ

### Порядок действий

1. Определение ошеломления (Скрытность против Наблюдательности)
2. Начало раунда
3. Ходы персонажей по Инициативе: **1 действие + 1 движение**

### Боевые проверки

- **Рукопашная:** Встречная проверка. Тот, у кого больше УУ, наносит удар
- **Стрелковая:** Простая проверка навыка

### Зона попадания

**Правило:** Инверсия результата d100 на попадание (например, выпало 37 → зона 73)

| Бросок d100 | Зона попадания |
|------------|----------------|
| 01-09 | Голова |
| 10-24 | Левая (ведомая) рука |
| 25-44 | Правая (ведущая) рука |
| 45-79 | Корпус |
| 80-89 | Левая нога |
| 90-00 | Правая нога |

### Урон

- **Урон:** Урон оружия + УУ встречной проверки
- **Нанесенные раны:** Урон - (Бонус Выносливости + Очки Доспеха)
- **Минимум:** 1 рана (если попал)

## 5. КРИТИЧЕСКИЕ РАНЫ И ЗАМИНКИ

### Критическая рана (Крит)
- **Условие:** Успех при выпадении дубля (11, 22, 33 и т.д.)
- **Эффект:** Противник получает критическую рану (бросок по таблице критических ран)

### Заминка (Fumble)
- **Условие:** Провал при выпадении дубля
- **Эффект:** Бросок по таблице заминок (01-100)

### Осечка (для порохового оружия)
- **Эффект:** Взрыв оружия
- **Урон:** Полный урон + единицы кубика попадания

---

## 6. МАГИЯ И МОЛИТВЫ

### Магия

- **Проверка сотворения:** Язык (магик)
- **Ингредиенты:** Снижают категорию ошибки (Крупная → Малая → Отмена)
- **Концентрация:** Длительная проверка. Если прервана → проверка Хладнокровия (-20) или малая ошибка
- **Рассеивание:** Встречная проверка Языка (магик) против сотворения

### Молитвы

- **Проверка:** Молитва (+0)
- **Гнев Богов:** Дубль ИЛИ единицы кубика ≤ Очков Греха
- **Благословения:** Эффекты за каждые 2+ УУ

---

## 7. ПСИХОЛОГИЯ

### Ужас
- **Проверка:** Хладнокровие
- **Провал:** Состояние "Сломлен" (Ранг ужаса + 1 за каждый -УУ)

### Страх
- **Эффект:** -1 УУ к проверкам рядом с источником
- **Ограничение:** Нельзя подойти ближе без проверки Хладнокровия

### Бешенство
- **Эффекты:**
  - Иммунитет к психологии
  - +1 атака
  - +1 бонус Силы
  - Нельзя отступать

## 8. СОСТОЯНИЯ

| Состояние | Эффект |
|-----------|--------|
| **Кровоточащий** | -1 рана в конце раунда. 10% шанс смерти при 0 ран |
| **Оглушенный** | Нет действий, движение в полсилы, -10 к проверкам |
| **Горящий** | 1d10 ран за раунд (минус Бонус Стойкости и доспех) |
| **Лежащий** | Движение только ползком. -20 к проверкам перемещения. Враги получают +20 к атаке |
| **Сломленный** | Бегство от врага. -10 к проверкам (кроме бегства) |
| **Отравленный** | -1 рана за раунд. -10 к проверкам |
| **Уставший** | -10 ко всем проверкам. Нужен отдых |

## 9. ЛЕЧЕНИЕ И СМЕРТЬ

### При 0 ран
- Состояние "Лежащий"
- Если не вылечен за количество раундов = Бонус Стойкости → "Бессознательный"

### Смерть
- **Условие:** 0 ран + "Бессознательный" + критических ран больше, чем Бонус Стойкости

### Исцеление
- **Хороший сон:** Проверка Выносливости (+20) восстанавливает УУ + Бонус Стойкости ран
- **Без проверки:** Восстановление ран = Бонус Стойкости

## 10. ПОРЧА

### Осквернение
- **Проверка:** Если Очков Скверны > (Бонус СВ + Бонус Выносливости)
- **Провал проверки Выносливости (+0):** Бросок по таблице физической или ментальной порчи

## 12. ЗАРАБОТОК

### Работа по профессии (Заработок)

- Персонаж работает неделю по своей карьере
- Делает **расширенную проверку рабочего навыка (+20)**
- Рабочий навык = выделенный *курсивом* навык 1-й ступени карьеры

**Результаты:**
- **Успех:** Бросок кубиков по статусу (см. ниже)
- **Провал:** Половина суммы
- **Ужасающий провал (-6):** Ничего не заработал или потерял всё

**Формула заработка по статусу:**
- **Медное сословие:** d10 × уровень статуса = медные пенни
- **Серебряное сословие:** d10 × уровень статуса = серебряные шиллинги
- **Золотое сословие:** d10 × уровень статуса = золотые кроны

**Примеры:**
- Медь 3: 3d10 мп (среднее ~16 мп, макс 30 мп)
- Серебро 2: 2d10 сш (среднее ~11 сш, макс 20 сш)
- Золото 1: 1d10 зк (среднее ~5.5 зк, макс 10 зк)

**Важно:** Это чистый доход с учётом всех расходов на неделю (еда, жильё, ремонт снаряжения).
