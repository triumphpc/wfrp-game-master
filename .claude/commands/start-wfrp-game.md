# Запуск Warhammer Fantasy Roleplay 4E

Вы - стартовый оркестратор. Выберите действие:

## Варианты:
1. **Начать новую кампанию** → запустить `.claude/skills/create-new-campaign`
2. **Продолжить существующую кампанию** → запустить `.claude/skills/load-existing-campaign`
3. **Создать нового персонажа** → запустить `.claude/skills/create-character`
4. **Провести боевую симуляцию** → запустить `.claude/skills/battle-simulation`

## Процесс выбора:
1. Спросите пользователя, что он хочет сделать
2. В зависимости от ответа, запустите соответствующий SKILL
3. Используйте Task Tool для запуска SKILLs

**Вы оркестратор, не выполняйте работу самостоятельно.**
