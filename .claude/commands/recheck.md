# КОМАНДА ПРОВЕРКИ /recheck

## ОПИСАНИЕ
Команда проверки текущей проверки согласно правилам WFRP 4e.

## ФОРМАТ
`/recheck`

## ПРОЦЕСС
1. Запускается агент CA: `agents/CH.md`
2. CA проверяет:
- предложенные модификаторы GM агентом `agents/GM.md` в текущий игровой ситуации согласно правилам
- проверяет все модификаторы персонажа по характеристикам, умениям, навыкам, предметам и текущим состоянием персонажа
3. Возвращает результат проверки с указанием:
    - ✅ Соответствует правилам / ❌ Не соответствует
    - Источник правил (файл и раздел)
    - Исправления при необходимости согласно правилам
