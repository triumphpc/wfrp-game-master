# Quickstart: Улучшение создания персонажей WFRP

## Prerequisites

- Go 1.21+
- Telegram Bot Token
- LLM провайдер (OpenAI/MiniMax)

## Quick Start

1. **Добавить команду /characters в handlers.go**

2. **Обновить GenerateCharacterMarkdown()** в character_creation.go для вывода русских характеристик

3. **Добавить обработку "сгенери имя"** в ProcessInput()

4. **Добавить детектор вопросов** - если ввод содержит "?" или ключевые слова, сделать запрос к LLM

## Commands to Implement

### /character
- Без аргументов: показать справку
- С аргументом: начать создание персонажа

### /characters
- Вывести список всех персонажей кампании

### "сгенери имя"
- Сгенерировать имя через LLM

## Testing

Тестирование производится вручную через Telegram:
1. Отправить `/help` - проверить наличие /character
2. Отправить `/character` - проверить справку
3. Отправить `/character Тест` - начать создание
4. На этапе характеристик спросить "как распределить?"
5. На этапе имени написать "сгенери имя"
6. Завершить создание - проверить карточку
7. Отправить `/characters` - проверить список
