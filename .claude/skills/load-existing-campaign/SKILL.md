---
name: load-existing-campaign
description: Загрузка существующей кампании для продолжения
---

## Phase 1: Выбор кампании
1. Показать список доступных кампаний из `history/`
2. Пользователь выбирает кампанию

## Phase 2: Загрузка состояния
1. Загрузить последнюю сессию
2. Загрузить карточки персонажей
3. Восстановить контекст

## Phase 3: Продолжение игры
1. Запустить GM агента с контекстом кампании
2. Начать с места остановки
