| Сложность | Модификатор |
| :--- | :--- |
| Очень легкая | +60 |
| Легкая | +40 |
| Средняя | +20 |

| Сложная | -10 |
| Трудная | -20 |
| Очень трудная | -30 |

## 2. ТИПЫ ПРОВЕРОК

### ПРОСТЫЕ ПРОВЕРКИ
- **d100 против умения**
- **Успех:** результат меньше или равен значению умения
- Используется для большинства стандартных проверок

### ДРАМАТИЧЕСКИЕ ПРОВЕРКИ
- Используются, когда важен **уровень успеха (УУ)**
- **УУ = разница между десятками** умения (после модификаторов) и десятками на кубике
- Пример: умение 65, бросок 42 → 6-4 = УУ 2

### ДЛИТЕЛЬНЫЕ ПРОВЕРКИ
- Требуют накопления определенного количества УУ за несколько бросков
- Используется для сложных задач (взлом замка, исследование)

### РАСШИРЕННЫЕ ПРОВЕРКИ
- Множественные броски для длительных задач
- Учет времени и ресурсов

## 3. ПОМОЩЬ И КООПЕРАЦИЯ

### ПРАВИЛА ПОМОЩИ
- **Один основной персонаж** совершает проверку
- **Каждый помощник дает +10** к проверке
- **Требования к помощнику:**
    - Должен иметь хотя бы 1 улучшение в умении
    - Должен находиться рядом (в зоне действия)
    - Должен иметь возможность помогать
- **Максимальное количество помощников:** обычно 2-3 (на усмотрение Мастера)

### ГРУППОВЫЕ ПРОВЕРКИ
- Все участники совершают проверку независимо
- Успех определяется по большинству или наилучшему результату
- Используется для скрытного перемещения, поиска и т.д.

## 4. УРОВЕНЬ УСПЕХА (УУ) - ФОРМУЛА РАСЧЕТА
УУ = (Десятки умения) - (Десятки броска)

text

**ПРИМЕРЫ:**
- Умение 65, бросок 42 → 6 - 4 = УУ 2 (успех)
- Умение 48, бросок 73 → 4 - 7 = УУ -3 (провал)

### ИНТЕРПРЕТАЦИЯ УУ:
- **УУ +1 или +2:** Незначительный успех
- **УУ +3 или +4:** Хороший успех
- **УУ +5 или +6:** Отличный успех
- **УУ +7+:** Феноменальный успех
- **УУ 0:** Граничный успех
- **УУ -1 или -2:** Незначительный провал
- **УУ -3 или -4:** Провал
- **УУ -5 или -6:** Ужасающий провал

## 5. СПЕЦИАЛЬНЫЕ СЛУЧАИ

### ДУБЛИ (КРИТИЧЕСКИЕ УСПЕХИ И ПРОВАЛЫ)
- **Дубль:** одинаковые цифры на десятках и единицах (11, 22, 33... 99, 00)
- **Критический успех:** дубль + успешная проверка
- **Критический провал (Заминка):** дубль + провал

### ПЕРЕБРОС ОЧКОВ СУДЬБЫ
- Можно потратить Очко Судьбы для переброска проверки
- Использовать новый результат (даже если хуже)
- Можно перебрасывать только свои проверки

### ВСТРЕЧНЫЕ ПРОВЕРКИ
- Два персонажа совершают проверку одного и того же навыка
- Сравнивается УУ
- Больший УУ побеждает

## 6. ШКАЛА СЛОЖНОСТИ ДЛЯ ИГРОКОВ

### ОБЪЯВЛЕНИЕ СЛОЖНОСТИ ПЕРЕД БРОСКОМ:

| Целевое число | Шанс успеха | Словесная оценка |
| :--- | :--- | :--- |
| **80+** | ~80%+ | Почти гарантированный успех |
| **60-79** | 60-80% | Очень высокие шансы |
| **40-59** | 40-60% | Хорошие шансы |
| **25-39** | 25-40% | Средние шансы |
| **15-24** | 15-25% | Сложно, низкие шансы |
| **10-14** | 10-15% | Очень сложно |
| **<10** | <10% | Почти невозможно |

## 7. МОДИФИКАТОРЫ ПРОВЕРОК (ОБЩИЕ)

### СИТУАЦИОННЫЕ ШТРАФЫ:
- **Туман/темнота:** -10 до -30
- **Движущаяся цель:** -10
- **Дальняя дистанция:** -10 до -30
- **Малая цель:** -10
- **Укрытие:** -10 до -40
- **Ранение:** -10 за каждую критическую рану

### СИТУАЦИОННЫЕ БОНУСЫ:
- **Помощь союзника:** +10 за каждого помощника
- **Хорошая позиция:** +10 до +20
- **Подготовка/время:** +10 до +20
- **Качество инструментов:** +10 до +30

## 8. БЫСТРЫЕ ПРАВИЛА ДЛЯ МАСТЕРА

### КОГДА ТРЕБОВАТЬ ПРОВЕРКУ:
- Действие рискованное или неочевидное
- Есть последствия провала
- Персонаж находится под давлением
- **НЕ ТРЕБОВАТЬ** на каждое действие (примерно каждые 3-5 действий)
