# Проверка правил WFRP 4E

## Input:
- query: вопрос о правилах
- context: контекст (опционально)

## Workflow:
1. **Parse query** - разобрать запрос на компоненты
2. **Search rules** - Task.run("checker-agent", "найди правило: {query}")
3. **Interpret** - интерпретировать правило в контексте
4. **Provide answer** - дать четкий ответ с цитатами

## Формат ответа:
Ответ на вопрос: [вопрос]
Найденные правила:
Интерпретация:
[Объяснение как правило применяется]

Пример применения:
[Конкретный пример]

## Special cases:
- Если правило противоречиво → запросить уточнение
- Если правило не найдено → предложить аналогии, но сказать что это аналогия
- Если нужна доменная экспертиза → запустить deeper analysis
