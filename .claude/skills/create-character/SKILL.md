# Создание персонажа WFRP 4E

## Workflow:
Подготовка → Выбор расы → Выбор класса → Характеристики → Карьера → Навыки → Таланты → Снаряжение → Финальная проверка

## Phase 1: Подготовка
1. Подготовить карточку нового персонажа, используя шаблон `.claude/skills/create-character/template.md`
- `chatacters/new.md`

## Phase 2: Выбор расы
.....


Для каждого этапа:

1. **GM предлагает вариант** → Task.run("gm-agent", "предложи выбор расы")
2. **Checker проверяет** → Task.run("checker-agent", "проверь правильность выбора расы")
3. **Logger записывает** → Task.run("logger-agent", "запиши выбор расы")

## Phase 3: Этапы создания
**Каждый этап требует подтверждения от Checker!**

1. Раса
2. Класс и карьера
3. Характеристики (броски/распределение)
4. Навыки и таланты
5. Снаряжение и богатство
6. История и внешность

## Phase 4: Финальная проверка
1. Запустить полную проверку через Checker
2. Сохранить персонажа
3. Создать backup
