# ШИРМА: БОЕВЫЕ ПРАВИЛА

## 1. ПОРЯДОК ДЕЙСТВИЙ В БОЮ

### ПРЕДБОЕВАЯ ФАЗА
1. **Определение ошеломления:**
    - Проверка Скрытности против Восприятия
    - Победитель получает преимущество "Ошеломление"
    - Может действовать до начала формального раунда

### НАЧАЛО РАУНДА
2. **Определение инициативы:**
    - Каждый персонаж бросает d10 + Инициатива
    - Действия по убыванию результата
    - При равенстве - бросок d100 (меньше побеждает)

### ХОД ПЕРСОНАЖА
3. **Действия в раунде:**
    - **1 основное действие** (атака, заклинание, молитва)
    - **1 движение** (перемещение на свою Скорость)
    - **Любое количество свободных действий** (говор, крик, смена оружия)

## 2. БОЕВЫЕ ПРОВЕРКИ

### РУКОПАШНЫЙ БОЙ
- **Встречная проверка:** Ближний бой атакующего против Ближнего боя защищающегося
- **Определение успеха:** Сравнение Уровней Успеха (УУ)
- **Победитель:** Тот, у кого больше УУ, наносит удар
- **Ничья:** Оба промахиваются

### СТРЕЛКОВЫЙ БОЙ
- **Простая проверка:** Баллистическое мастерство
- **Модификаторы:** Дистанция, укрытие, движение цели
- **Успех:** Попадание, расчет урона

## 3. ЗОНЫ ПОПАДАНИЯ

### ТАБЛИЦА ЗОН ПОПАДАНИЯ

| Бросок d100 | Зона попадания | Описание |
| :--- | :--- | :--- |
| **01-09** | **Голова** | Череп, лицо, шея |
| **10-24** | **Левая рука** (ведомая) | Плечо, локоть, кисть |
| **25-44** | **Правая рука** (ведущая) | Плечо, локоть, кисть |
| **45-79** | **Корпус** | Грудь, живот, спина |
| **80-89** | **Левая нога** | Бедро, колено, ступня |
| **90-00** | **Правая нога** | Бедро, колено, ступня |

### ПРАВИЛА ОПРЕДЕЛЕНИЯ:
- **Инверсия результата:** 37 → зона 73 (Корпус)
- **При стрельбе:** Определяется по таблице
- **В рукопашной:** Можно выбрать зону с -20 к проверке

## 4. РАСЧЕТ УРОНА И РАН

### ФОРМУЛА УРОНА:
Базовый урон = Урон оружия (например: +РС+4)
Дополнительный урон = УУ встречной проверки (в рукопашной)
Итоговый урон = Базовый + Дополнительный

### ФОРМУЛА РАН:
Нанесенные раны = Урон - (Бонус Стойкости + Очки Доспеха)
Минимум: 1 рана (даже если защита превышает урон)

**ПРИМЕР:**
- Урон: 12
- Бонус Стойкости: 3
- Доспех: 2 AP
- Раны: 12 - (3 + 2) = 7 ран

## 5. ПРЕИМУЩЕСТВО (ADVANTAGE)

### ПОЛЬЗА ПРЕИМУЩЕСТВА:
- **Каждое преимущество дает +10** к боевым или психическим проверкам
- **Накопительный эффект:** 2 преимущества = +20

### КАК ПОЛУЧИТЬ ПРЕИМУЩЕСТВО:
1. **Атака ошеломленного врага**
2. **Использование натиска** (специальное действие)
3. **Победа во встречной проверке**
4. **Нанесение ран без вступления в бой** (стрельба, засада)
5. **Критическое попадание**
6. **Успешный тест на страх/ужас** против врага

### КАК ПОТЕРЯТЬ ПРЕИМУЩЕСТВО:
1. **Провал встречной проверки**
2. **Получение любого состояния** (оглушен, ранен и т.д.)
3. **Получение раны** (любой тяжести)
4. **Провал теста на страх/ужас**
5. **Конец боя** (все преимущества сбрасываются)

### МАКСИМАЛЬНОЕ ПРЕИМУЩЕСТВО:
- **Обычно:** максимум 5 преимуществ
- **Специальные таланты:** могут увеличить лимит

## 6. КРИТИЧЕСКИЕ ПОПАДАНИЯ

### УСЛОВИЯ КРИТА:
1. **Успешная атака** (попадание)
2. **Дубль на кубиках** (11, 22, 33... 99, 00)
3. **Или результат делится на 10** (для оружия с качеством "Пронзающее")

### ЭФФЕКТ КРИТА:
- **Противник получает критическую рану**
- **Бросок по таблице критических ран** (1d100 + урон)
- **Эффект зависит от зоны попадания**

### КРИТИЧЕСКИЕ РАНЫ:
- **Накладывают состояния** (перелом, кровотечение и т.д.)
- **Дают штрафы** к характеристикам и проверкам
- **Требуют лечения**
- **Могут быть смертельными**

## 7. ЗАМИНКИ (FUMBLE)

### УСЛОВИЯ ЗАМИНКИ:
1. **Провал атаки** (промах)
2. **Дубль на кубиках** (11, 22, 33... 99, 00)

### ЭФФЕКТ ЗАМИНКИ:
- **Бросок по таблице заминок** (1d100)
- **Эффекты:** потеря оружия, падение, самоповреждение
- **Для порохового оружия:** осечка (см. ниже)

### ОСЕЧКА ПУШЕЧНОГО ПОРОХА:
- **Для порохового оружия** при заминке
- **Взрыв оружия**
- **Урон владельцу:** Полный урон оружия + единицы куба попадания
- **Проверка Удачи:** для уменьшения урона

## 8. СПЕЦИАЛЬНЫЕ БОЕВЫЕ ДЕЙСТВИЯ

### НАТИСК (CHARGE):
- **Действие:** Движение + Атака
- **Требование:** Минимум 4 ярда разбега
- **Бонус:** +10 к атаке, +1 УУ к урону
- **Риск:** Открытая защита

### ОТСТУПЛЕНИЕ:
- **Действие:** Движение от врага
- **Требование:** Проверка Инициативы (-20)
- **Эффект:** Безопасный отход
- **Риск:** Свободная атака при провале

### ЗАЩИТА (ПАРИРОВАНИЕ):
- **Действие:** Отказ от атаки
- **Эффект:** +20 к Ближнему бою до следующего хода
- **Можно использовать** один раз за раунд

### БРОСОК НА ЗЕМЛЮ:
- **Действие:** Атака с целью сбить с ног
- **Проверка:** Встречная проверка Силы или Ближнего боя
- **Эффект:** Противник получает состояние "Лежащий"

## 9. СТРЕЛКОВОЕ ОРУЖИЕ

### МОДИФИКАТОРЫ СТРЕЛЬБЫ:

| Ситуация | Модификатор |
| :--- | :--- |
| **Короткая дистанция** (½ дальности) | +10 |
| **Дальняя дистанция** (макс. дальность) | -10 |
| **Очень дальняя** (за пределами) | -20 |
| **Движущаяся цель** | -10 |
| **Малая цель** (меньше человека) | -10 |
| **Цель в укрытии** | -10 до -40 |
| **Туман/темнота** | -10 до -30 |
| **Прицеливание** (трата действия) | +10 |
| **Стрельба очередью** (арбалет/ружье) | -10 |

### ПЕРЕЗАРЯДКА:
- **Лук:** свободное действие
- **Арбалет:** 1 действие
- **Ружье/пистолет:** 2 действия

## 10. БОЕВЫЕ СОСТОЯНИЯ (КРАТКО)

### ВЛИЯНИЕ НА БОЙ:
- **Оглушенный:** Нет действий, -10 к проверкам
- **Лежащий:** -20 к проверкам, +20 к атаке противника
- **Кровоточащий:** -1 рана/раунд
- **Горящий:** 1d10 урона/раунд

### СНЯТИЕ СОСТОЯНИЙ:
- **Оглушенный:** 1 раунд отдыха
- **Лежащий:** действие "Встать"
- **Кровоточащий:** проверка Лечения
- **Горящий:** действие "Потушить"

## 11. БЫСТРЫЕ ПРАВИЛА ДЛЯ МАСТЕРА

### БАЛАНС БОЯ:
- **Нормальная сложность:** 1-2 противника на персонажа
- **Сложный бой:** 3 противника на персонажа
- **Очень сложный:** 4+ противника или босс

### ИНИЦИАТИВА NPC:
- Группы NPC могут действовать вместе
- Использовать среднюю инициативу
- Описывать действия группы, а не каждого

### УПРОЩЕННЫЙ УРОН ДЛЯ NPC:
- **Слабые враги:** урон = РС + 2
- **Средние враги:** урон = РС + 4
- **Сильные враги:** урон = РС + 6

### СМЕРТЬ NPC:
- **Мобы:** умирают при 0 HP
- **Важные NPC:** могут сдаться, бежать или получить критические раны
- **Боссы:** специальные правила выживания
