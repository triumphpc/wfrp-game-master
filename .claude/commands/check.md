# КОМАНДА ПРОВЕРКИ /check

## ОПИСАНИЕ
Команда для проверки соответствия правил WFRP 4e.

## ФОРМАТ
`/check [текст для проверки]`

## ПРИМЕРЫ
`/check персонаж атакует длинным луком урон 1d10+4`
`/check развитие силы с 41 до 46 за 50 опыта`
`/check проверка скрытности со сложностью +0`

## ПРОЦЕСС
1. Запускается агент: `.claude/agents/CH.md`
2. Агент анализирует текст через RAG-MCP-SERVER
3. Возвращает результат проверки с указанием:
    - ✅ Соответствует правилам / ❌ Не соответствует
    - Источник правил (файл и раздел)
    - Исправления при необходимости
