# Создание новой кампании WFRP 4E

## Workflow:
- Настройка компании -> Создание персонажей -> Настройка начальной сцены -> Запуск игры

## Phase 1: Настройка кампании
1. Спросить название кампании `[campaign_name]`
2. Создать директорию: `history/[campaign_name]/`
3. Создать поддиректории: `characters/`

## Phase 2: Создание персонажей
- Спросить у игрока: Создать нового персонажа или использовать текущего?
- Если уже создал или выбрал текущего, то добавить выбор "Продолжить настройку сцены кампании" (Phase 3: Настройка начальной сцены)

**Выбирал Новый персонаж**
- Запустить SKILL `.claude/skills/create-character/SKILL.md`
- После создания скопировать созданную карточку из `history/characters/[character_name]` -> `history/[campaign_name]/characters/[character_name]` текущей кампании
- Вернуться к Phase 2: Создание персонажей

**Выбрал Текущий персонаж**
- Вывести список текущих персонажей из `history/characters/` и спросить у пользователя какого он выбирает
- Скопировать выбранную карточку из `history/characters/[character_name]` -> `history/[campaign_name]/characters/[character_name]` текущей кампании
- Вернуться к Phase 2: Создание персонажей


## Phase 3: Настройка начальной сцены
1. Смоделировать сценарий новой сессии в духе WHFR 4ред. (`rules/01_введение.md`)

## Phase 4: Запуск игры
1. Передать вводные по персонажам и сценарию `.claude/agents/gm.md`
