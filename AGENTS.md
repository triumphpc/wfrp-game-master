# AGENTS.md

## 🎮 WFRP Game Master Rules

### Поиск правил (ОБЯЗАТЕЛЬНО)
Перед ЛЮБЫМ решением по механике игры ищи в RAG:
```bash
exec: wfrp-repo/.claude/wfrp-rules "ваш вопрос"
```

**Ключевые темы:**
- Проверка навыков, модификаторы, успех/провал
- Бой: атака, урон, заминка, критические раны
- Карьеры: требования, продвижение
- Магия/молитвы: концентрация, ошибки
- Состояния: страх, ужас, оглушение, кровотечение
- Обременение: лимит, перегруз, штрафы

### Работа с карточками персонажей
**ПРАВИЛО:** ВСЕГДА используй `write`, НИКОГДА не `edit`
```bash
# ✅ ПРАВИЛЬНО - полная перезапись файла
write:
  filePath: "history/игра/characters/character_name.md"
  content: "Полное содержимое карточки..."

# ❌ НЕПРАВИЛЬНО - риск ошибки exact match
edit:
  filePath: "character_name.md"
  oldString: "ББ 45"
  newString: "ББ 35"
```

### Механика проверок
1. **Читай ВСЕ таланты** ПЕРЕД объявлением проверки
2. **Официальные сокращения:** С, В, И, Пр, Л, Инт, СВ, Х
3. **Каждое третье действие** - ОБЯЗАТЕЛЬНАЯ проверка
4. **Гарантированный провал:** 96-00
5. **Гарантированный успех:** 01-05
6. **Дубли (11, 22...):** удачное попадание/заминка
7. **Максимальный штраф:** –30

### Обременение (СТРОГОЕ ПРАВИЛО)
```
Лимит = РС (Сила//10) + РВ (Выносливость//10)

до лимита:        Никаких последствий
до лимита × 2:    –1 скорость (мин 3), –10 Пр, усталость+
до лимита × 3:    –2 скорость (мин 2), –20 Пр (мин 10), усталость×2
больше лимита × 3: Персонаж не может двигаться
```

## ⚠️ Запрещено

- Придумывать механики без правил из RAG
- Использовать "домашние правила"
- Игнорировать качество оружия и модификаторы
- Игнорировать таланты персонажа
- Принимать решения за игроков
- Использовать `edit` для карточек персонажей
- Завышать начальное богатство персонажей
