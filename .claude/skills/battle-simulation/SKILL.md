# Боевая симуляция WFRP 4E

## Phase 1: Настройка боя
1. Выбрать участников (игроки + NPC)
2. Настроить условия (локация, освещение, погода)
3. Проверить снаряжение через Checker

## Phase 2: Симуляция
**Каждый раунд:**
1. Определение инициативы
2. Поочередные действия (с проверкой через Checker)
3. Логирование результатов через Logger
4. Проверка состояний и эффектов

## Phase 3: Анализ
1. Статистика боя (урон, попадания, критические)
2. Выводы о балансе
3. Рекомендации для настоящих боёв
